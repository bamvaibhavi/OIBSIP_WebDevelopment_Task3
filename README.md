# Task 3: Basic To-Do Webapp
A daily task management application titled "TaskMaster".

## Features
* **Task Management:** Users can add, edit, and delete tasks.
* **Categorization:** Separate sections for "Pending Tasks" and "Completed Tasks".
* **Task Details:** Automatically records the date and time when a task is added.
* **Status Updates:** Allows users to mark tasks as "complete," moving them to the finished list.
